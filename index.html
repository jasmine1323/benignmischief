About This Project

This is a demonstration of sustainable web design principles. By returning to basics and eliminating unnecessary complexity, we can create websites that consume a fraction of the energy used by modern sites while remaining fully functional and accessible.

The modern web has become bloated with unnecessary JavaScript frameworks, high-resolution images, tracking scripts, and advertising networks. A typical website now weighs over 2 MB and makes hundreds of requests to third-party servers. This approach is neither sustainable nor necessary.

Design Principles

Static HTML with minimal CSS | No JavaScript required | Default system fonts | Dithered, compressed images | Optimized for low-bandwidth connections | Accessible to older devices and assistive technologies
